#Сказочный бот#

Благодаря самообучающимся алгоритмам искусственного интеллекта, это самый качественный голосовой синтез русскоязычного текста. Бот разработан студентами ВШЭ: *@yanixrm, @sergebolgar, @saocybacon, @aksrrr*.
